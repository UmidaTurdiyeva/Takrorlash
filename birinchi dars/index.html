Git

Git - bu kodlarni nazorat qilish texnologiyasi bo'lib, lokal tarmoqda barcha yozilgan kodlarni saqlaydi

Git texnologiyasida dasturchi yaratayotgan loyihasini bir necha versiyalarga bo'lib uni nazorat qilishi mumkin

Endi Git texnologiyasining asosiy buyruqlarini ko'rib chiqamiz:
  • git add . - o'zgarishlarni aniqlash
  • git commit -m" message" - aniqlangan o'zgarishga nom berish
  • git status - barcha commitlarni ko'rsatish
  • git diff -
  • git log -
  • git reset -
  • git checkout -
  • git init - yaratilgan omborga (repository) kirish
  • git branch - yangi filial yaratish
  • git pull -
  • git push -
  • git restore - o'zgarishlarni bekor qilish

GitHub

GitHub - bu yozilgan kodlarni onlayn ma'lumotlar omborida ya'ni cloud da saqlash imkonini beruvchi plarforma

GitHub yordamida bir necha dasturchi bir loyiha ustida bir vaqtda ishlashi mumkin.

HTML

1. Html – bu …
Veb sahifalarni yaratish uchun ishlatiladigan standart gipermatnli belgilash tili
2. Html bu ..
Veb sahifaning strukturasini tasvirlovchi hujjat
HTML elementlar qatoridan iborat
HTML elementlari yordamida brouzerga kotentlarni hohlaganimizday joylashtira olamiz
HTML elementlari yordamida veb sahifadagi ma’lumotlarni ifodalashimiz mumkin, masalan, bu sarlavha (head), bu paragraph (p), bu rasm (img), va hakozo.
HTML elementining o’zi quyidagi ko’rinishda bo’ladi:

< boshlovchi teg> content </yakunlovchi teg>

Elementlar ochiq yoki yopiq, toq yoki juft bo’lishi mumkin:
<br>,< hr> va …
HTML hujjatlarining barchasi gekloratsiyasidan boshlanishi kerak
HTML tegi hisoblanmaydi
U brauzer uchun ma’lumot bo’lib, u hujjatning tipi bildiradi
