input (kiritilgan ma'lumot) elementi

  • HTMLda < input > elementi foydalanuvchidan ma'lumot olish uchun veb-formalarda interaktiv boshqaruv elementlarini yaratish uchun ishlatiladi;
  • kirish ma'lumotlari turlari va atributlarining mumkin bo'lgan kombinatsiyalarining ko'pligi tufayli u eng kuchli va murakkab HTML elementlaridan biridir.
  • input elementi ochiq (toq) teg hisoblanadi < input >
  • input elementi default xolda 20ta belgi sig'adigan kenglikda bo'ladi:
INPUT TYPES

label (yorliq, belgi) elementi

  • HTML <label> elementi foydalanuvchi interfeysi elementi uchun yorliqni ifodalaydi.
  • labelni inputga for atributi bilan bog'lamiz.
  • labeldagi for atributi bilan inputdagi id atributi qiymati bir xil bo'lishi kerak.
  • label elementi juft element bo'lib, <label > ochiq tegi, content qismi va < /label> yopiq tegidan iborat, ya'ni
  • <label> Username </label>

SELECT (tanlash) elementi

  • Html da select elementi foydalanuvchi tanlashi uchun bir nechta option (variant) lar taqdim etadi.
  • option elementida tanlash mumkin bo'lgan variantlar yoziladi.
  • Default holatda select da birinchi element tanlangan bo'ladi.
  • select da berilgan optionlardan qaysidiri oldindan tanlangan bo'lishi uchun selected atribitidan foydalaning
  • Default holatda select da bitta option ko'rinadi. Ko'rinadigan optionlar sonini size atributi orqali o'zgartirsak bo'ladi.
  • Select elementida birnechta qiymatni tanlash uchun multiple atributidan foydalaning. Bunda select elementida bir nechta option (variant) larni tanlash mumkin.

TEXTAREA (matn maydoni) elementi

  • textarea elementi katta hajmdagi matnlarni kiritish uchun ishlatiladi.
  • TextArea da kiritilgan teks bir qatordan ko'p bo'lishi kumkin.
  • rows atributi textarea da ko'rinib turadigan (boshlang'ich) qatorlar soniga javob beradi. Ya'ni rows ni boshlang'ich height (balandlik) desak ham bo'ladi.
  • cols attributi textarea da ko'rinib turadigan (boshlang'ich) ustunlar soniga javob beradi. Ya'ni cols ni boshlang'ich width (kenglik) desak ham bo'ladi.

Qo'shimcha elementlar

  • button (tugma) elementi bosiladigan knopkani bildiradi.
  • fieldset (maydonlar to'plami) elementi formada elementlarni guruhlash uchun ishlatiladi.
  • legend (надпись, yozuv) elementi fieldset lar uchun ramka sifatida ishlatiladi.
  • datalist (ma'lumotlar ro'yxati) elementi input elementi uchun oldindan tayyorlangan variantlar ro'yxatini ko'rsatadi.
  • input elementining list atributi datalist elementining id atributiga bir xil bo'lishi kerak.
