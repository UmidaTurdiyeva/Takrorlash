Action (harakat) atributi

Action atributi forma jo'natilganda (submit qilinganda) ma'lumotlar borishi kerak bo'lgan server manzili (url) yoziladi.


< form action="/action_page.php" >

Target(maqsad) atributi

Target atributi forma jo'natilgandan so'ng olingan javobni qayerda ko'rsatilishi kerak ekanligini bildiradi.


value Description
_blank Javob boshqa yangi sahifada paydo bo'ladi.
<form action="url manzil" target="_blank">
_self Javob hozirgi oynada paydo bo'ladi.
< form action="url manzil" target="_self" >
_parent Javob dastlabki oynada paydo bo'ladi.
<form action="url manzil" target="_parent">
_top Javob to'liq oynada paydo bo'ladi.
<form action="url manzil" target="_top">
framename Javob iframe elementi ichida bo'ladi.
< form action="url manzil" target="framename">

Metod atributi

Metod atributi jo'natish tugmasi bosilgandan so'ng shu forma HTTP ga qaysi metod orqali ma'lumot jo'natilishini ko'rsatadi.

Bu metodlar 2 xil:

  • get metodi

    • GET ma'lum bir manbadan ma'lumotlarni so'rash uchun ishlatiladi.
    GET so'rovlari hech qachon nozik ma'lumotlar bilan ishlashda ishlatilmasligi kerak.
    GET so'rovlarida uzunlik cheklovlari mavjud. (2048 ta belgi)
    Запросы GET используются только для запроса данных (не для изменения).
    GET so'rovlari brauzer tarixida qoladi.
    GET so'rovlariga xatcho'p qo'yish mumkin
    GET so'rovlarini keshlash mumkin

  • post metodi

    • POST resurs yaratish/yangilash uchun serverga ma'lumotlarni yuborish uchun ishlatiladi.
    POST so'rovlari hech qachon keshda saqlanmaydi
    POST so'rovlari brauzer tarixida saqlanmaydi
    POST so'rovlarini xatcho'p qilib bo'lmaydi
    POST so'rovlarida ma'lumotlar uzunligi bo'yicha cheklovlar yo'q.
